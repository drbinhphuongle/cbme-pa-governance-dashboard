# CBME Programmatic Assessment Governance + Dashboard Kit

A minimal-data, decision-oriented governance and analytics toolkit for programmatic assessment in CBME.
Includes governance pack, decision rules, CCC templates, a lightweight dashboard, and a demo dataset.

## Whatâ€™s inside
- Governance Pack: principles, roles/RACI, cadence, due process, data quality & privacy
- Decision Rules: minimum evidence standards, triggers (green/amber/red), outlier handling
- CCC Templates: minutes template and decision documentation
- Dashboard Kit: Learner Snapshot, EPA Progress, Quality & Fairness
- Demo Dataset + Data Dictionary: reproducible simulated dataset for demonstration

## Repo structure
- /paper: manuscript + figures
- /toolkit: governance pack + templates + user guide
- /dashboard: Excel + Google Sheets link
- /data: demo dataset + data dictionary
- /docs: implementation guide + documentation

## Quick start
1. Open `/dashboard/excel/PA-Gov_DashboardKit_v1.0.0.xlsx`
2. Load `/data/demo-dataset/DemoDataset_v1.0.0.csv`
3. Review outputs in the 3 dashboard views
4. Use `/toolkit/templates/CCC_Minutes_Template_v1.0.0.docx` for CCC decisions
